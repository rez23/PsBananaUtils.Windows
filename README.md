# PsBananaUtils.Windows
This is a part of [PsBananaUtils](https://github.com/rez23/PsBananaUtils) module.
# Install the module
Install the module via Install-Module or Install-PsResource
`powershell
# Install the module
Install-PsResource -Name PsBananaUtils.Windows -TrustedRepo -Scope CurrentUser

# Then import in your session
Import-Module PsBananaUtils.Windows
`
